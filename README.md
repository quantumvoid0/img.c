# img.c
A lightweight gtk based image viewer

## Installation

```fish
git clone https://github.com/quantumvoid0/img.c
cd img.c
make install
```

## Uninstallation

```fish
git clone https://github.com/quantumvoid0/img.c
cd img.c
make uninstall
```

## Usage

just run `img /path/to/img` in ur terminal or set `img` as default opener in ur file manager
